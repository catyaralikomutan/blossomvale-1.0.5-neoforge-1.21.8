# BlossomVale

BlossomVale — NeoForge için pastel-çiçek temalı ağaç ve biyom modu.

## Kısa (TR) / Short (EN)
TR: BlossomVale, Minecraft 1.21.8 (NeoForge) için geliştirilmiş; yeni fuşya odun ailesi, pastel pembe korular (Fuchsia Thickets) ve keşif odaklı mekanikler ekler. Yapraklar vanilla davranışı gösterir, bazen XP serper ve fidan/çubuk/nadir elma bırakır.

EN: BlossomVale adds pastel-pink groves and a complete fuchsia wood family to Minecraft 1.21.8 (NeoForge). Leaves behave like vanilla (decay, saplings/sticks/rare apples) and sometimes release XP when broken.

## Özellikler / Features
- Tam fuchsia odun ailesi: log, stripped log, planks, leaves, sapling, door/trapdoor, stairs, slab, fence, gate, button
- Dengelenmiş alet katmanı: demirden güçlü, elmasın biraz üzerinde
- Yeni biyom: Fuchsia Thickets — yeni chunk'larda ve yeni dünyalarda üretim
- Yaprak davranışı: vanilla çürüme, nadir elma ve fidan/çubuk düşüşü; kırınca bazen XP verir (petal‑wisps)
- MCreator ile oluşturuldu (ilk sürüm: derlenmiş JAR; kaynak ileride paylaşılabilir)

## Sürüm / Version
- Mod sürümü: 1.0.5
- Minecraft: 1.21.8
- Loader: NeoForge (Forge uyumlu, build/test NeoForge üzerinde yapılmıştır)
- Beklenen JAR adı: blossomvale-1.0.5-neoforge-1.21.8.jar

## Kurulum / Installation
1. NeoForge (veya uyumlu Forge sürümünü) yükleyin.
2. `mods` klasörüne `blossomvale-1.0.5-neoforge-1.21.8.jar` dosyasını koyun.
3. NeoForge profiliyle Minecraft'ı başlatın.
